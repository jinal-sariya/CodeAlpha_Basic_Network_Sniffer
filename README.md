# 🛡️ Network Packet Sniffer

A simple Network Packet Sniffer built using Python and Scapy.

## Features

- Capture live network packets
- Display Source IP
- Display Destination IP
- Detect Protocol (TCP, UDP, ICMP)
- Show Packet Payload
- Lightweight and easy to use

## Technologies Used

- Python
- Scapy

## Installation

Clone the repository

```bash
git clone https://github.com/JINAL SARIYA/CodeAlpha_Network_Sniffer.git
```

Go to project folder

```bash
cd CodeAlpha_Network_Sniffer
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python packet_sniffer.py
```
